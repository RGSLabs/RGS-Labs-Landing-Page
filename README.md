## Stack tecnológico

Construido de la misma forma en que se construye cada proyecto de RGS Labs™: **sin build tools, sin frameworks, sin dependencias que mantener.**

- **HTML5 / CSS3 / JavaScript** puro — un solo `index.html` autocontenido
- **Canvas API** para la animación de red de partículas del hero
- **IntersectionObserver** para las animaciones de scroll-reveal
- CSS `position: sticky` para la sección con scroll fijo
- Hosteado en **GitHub Pages**, servido directamente desde este repositorio
- Google Fonts (Space Grotesk + Inter + IBM Plex Mono) vía CDN
- Datos estructurados Schema.org + metadatos Open Graph para SEO
- Desarrollado con **asistencia de IA**, dirigido y revisado en cada paso por un humano

Sin React, sin pipeline de build, sin `node_modules`. Se clona, se abre `index.html`, y funciona — exactamente el tipo de entrega de bajo mantenimiento que RGS Labs™ le da a cada cliente.

## Principios de diseño detrás de esta construcción

- **Claridad sobre complejidad** — cada sección tiene un solo propósito
- **Solo contenido real** — sin Lorem Ipsum, sin testimonios inventados, sin portafolio de foto de stock; todo lo que se muestra es un proyecto real y entregado
- **Mobile-first y accesible** — respeta `prefers-reduced-motion`, navegable por teclado, HTML semántico
- **Precios transparentes** — qué incluye y qué no incluye cada servicio, declarado desde el inicio
